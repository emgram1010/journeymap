## Washer/Dryer White-Glove v1 Pilot

This folder contains the first workbook-ready pilot derived from `journey-maps/drafts/washer-dryer-white-glove-v1-draft.md`.

### Google Sheets tab mapping
- `journey_map.csv` -> `Journey Map`
- `stage_detail.csv` -> `Stage Detail`
- `stakeholders.csv` -> `Stakeholders`
- `journey_scenarios.csv` -> `journey_scenarios`

### Companion artifact
- `anti_journey_map.md` -> first domain-specific anti-journey example for this use case

### Confidence note
- Stages 1-2 are the strongest.
- Stages 3-8 are useful pilot content but still include draft synthesis and open questions.

### How to use
1. Create a new Google Sheet.
2. Import each CSV into its own tab using the tab names above.
3. Use the anti-journey map as the failure/exception companion layer.
