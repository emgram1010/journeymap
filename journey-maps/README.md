## Journey Maps

Use this folder for client-specific journey assets and draft/source material.

### Structure
- `drafts/` -> reference drafts, archived source material, and example inputs
- `pilots/` -> repo-tracked pilot packages with CSV exports and companion artifacts

### Rule of thumb
- if it is reusable template logic, it belongs in `workspace/templates/`
- if it is raw/reference journey material, it belongs in `journey-maps/drafts/`
- if it is a client/pilot deliverable or export, it belongs in `journey-maps/pilots/`
